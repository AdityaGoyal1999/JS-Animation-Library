About

The library is a front end library where the user can eaily connect multiple backgrounds from the provided pallete to one's webpage and make it more presentable.
Moreover, this documentation contains robust information with examples for getting started.

Initializing the library

  • Make sure that library.js is present in your project.
  • Create a javascript file which is connected to your webpage where you want the library to perform its functions.
  • In the javascript file, when the contents of the attached HTML file is loaded, write the following code:
    • const bg = new BackgroundGenerator();
